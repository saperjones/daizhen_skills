# daizhen_skills

daizhen 的个人 Claude Code 技能集（plugin + marketplace）。

技能：`ai_reply`、`dz_spec`、`oral_reply`、`prd_design`、`review_format_spec`、`tech_doc_plain`。

## 在另一台电脑安装

```
/plugin marketplace add saperjones/daizhen_skills
/plugin install daizhen@daizhen
```

装完重启 Claude Code，技能即以 `daizhen:<name>` 出现。

## 更新

```
/plugin marketplace update daizhen
```
